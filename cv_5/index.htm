JavaScript vs JQuery

jQuery is a fast, small, and feature-rich JavaScript library. It makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers. With a combination of versatility and extensibility, jQuery has changed the way that millions of people write JavaScript. [1], 24.3.2021

Most common way how to work with JQery library is:

  1. Link external JQery library or download it and link locally
  2. Select and manipulate DOM elements
  3. Handle events
  4. Write function and optionaly asociate them to events
  5. Work with canvas or use JavaScript API
  6. Implement AJAX
